The back-end of the Blue Sky Flights Management System is built with Node.js and Express.js, following the MVC architecture for structured code management. It handles user authentication, flight data processing, ticket purchases, and API communication. Using Express Routers, the system efficiently manages requests, while dotenv secures sensitive data. Server-side logging and error handling ensure reliability, and cookies are used for session management
