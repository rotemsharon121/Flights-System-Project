אודות - באקאנד 🛠️

צד השרת של מערכת ניהול הטיסות נבנה באמצעות node.js ו-express.js, תוך שימוש בארכיטקטורת MVC לניהול קוד מסודר. הוא מטפל באימות משתמשים, עיבוד נתוני טיסות, רכישת כרטיסים ותקשורת API. באמצעות Express Routers, המערכת מנהלת בקשות בצורה יעילה, בעוד dotenv מאבטח נתונים רגישים. לוגים בצד השרת וניהול שגיאות מבטיחים אמינות, ועוגיות משמשות לניהול סשנים

English

צד שרת

📚 תשתיות ופיתוח

  • 🚀 שימוש ב- Node.js ו- Express.js
    מאפשר פיתוח שרת יעיל ומהיר עם מסגרת קלה לניהול בקשות ותגובות
  • 🔀 שימוש ב- Express Routers לניתוב נכון של בקשות
    מבטיח ארגון מסודר של הנתיבים ומקל על ניהול הנתונים והבקשות
  • 🏗️ ארכיטקטורת MVC לניהול קוד מודולרי ומסודר
    מחלק את הקוד בצורה ברורה למודלים, תצוגות ובקרות לשיפור התחזוקה והקריאות
  • 🔒 שימוש ב- dotenv לשמירה בטוחה של נתונים רגישים
    מחלק את הקוד בצורה ברורה למודלים, תצוגות ובקרות לשיפור התחזוקה והקריאות
  • 🔄 nodemon לסביבת פיתוח נוחה ועדכנית
    מרענן את השרת אוטומטית עם כל שינוי בקוד, מה שמייעל את תהליך הפיתוח

📢 ניהול תהליכים ושגיאות

  • 📣 לוגים בטרמינל
    הצגת הודעות על כל פעולה של משתמש: כניסה לדפים, חיפושי נתונים, צפייה בלקוחות, הוספת לקוחות, שינוי פרטי לקוחות, מחיקת לקוחות, רכישת כרטיסים ומחיקתם
  • ⚠️ התראות שגיאות בשרת
    הצגת הודעות על תקלות ושגיאות מערכת

🔒 אבטחה וניהול משתמשים

  • 🔐 Cookies לניהול משתמשים
    ללא הצפנה (לצורך הדגמה בלבד)
  • 📚 API לניהול האתר
    יצירה ושימוש ב- REST API לצורך הצגה, חיפוש, שינוי ומחיקה של נתונים באמצעות fetch
  • 💼 מערכת התחברות וניהול משתמשים
    זיהוי משתמשים רשומים ואימות זהות
  • 📷 ניהול תמונות פרופיל
    תמונות נשמרות בתיקיית הפרויקט, והנתיב שלהן נשמר בבסיס הנתונים
  • 🗑️ מחיקת משתמש
    בעת מחיקת משתמש, התמונה המתאימה נמחקת מהשרת
