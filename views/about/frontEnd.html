About - Front end ✨

The front-end of the Blue Sky Flights Management System provides a responsive and user-friendly experience using Bootstrap for styling. It features flight searches, ticket purchases, an airline directory, and user profile management, The interface ensures smooth navigation with well-organized tables, filters, and a modern design

Hebrew / עברית

Client Side

🎨 Design and User Experience

  • 💠 Bootstrap
    Using Bootstrap for website design and improving user experience -
    Provides responsive and consistent design for all website pages.
  • 📥 File Downloads
    Option to download task forms and configuration files -
    Allows downloading the original task form, database configuration files, and error handling files.
  • 🖥️ Displaying Data from API
    Importing and displaying dynamic information directly from the backend API -
    Presenting data in an organized and user-friendly manner using styled tables with search, sort, and filter options.

📊 Key Pages

  • ✈️ Flights Page
    Displaying all flights in the database with the option to purchase tickets based on the logged-in user.
  • 🔍 Flight Search
    Search for flights by origin country, destination country, flight date, and landing date, with the option to purchase tickets directly from the search.
  • 🌎 Airline List
    Displaying all airlines in the database.
  • 📓 Search by Airline
    Search for flights by airline name, airline number, manager name, or the country where it operates.
  • 👤 Customer Management
    A page displaying information about all registered customers in the system.

💻 User Management

  • 👁 Personal User Page
    Edit profile, view and delete purchased flight tickets, and delete the profile.
  • 🔐 Registration/Login Page
    Allows searching for flights by origin country, destination country, flight date, and landing date, with the option to purchase tickets directly from the search.
  • 📋 Connected User Identification
    Identifying the connected user using cookies for personal purchases.
  • 🌟 Ticket Purchase Management
    Purchasing tickets for the connected user only.
